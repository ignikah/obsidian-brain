# AIAP 練習系統

## 原則

- 官方公告試題是主要練習材料。
- 筆記不全文複製題目，只記錄題號、考點、錯因與修正理由。
- CKO 依錯題考點出原創變形題，避免只背答案。

## 每份試題處理流程

1. 第一次作答：計時，不查資料。
2. 對答案：只標出錯題與猶豫題。
3. 錯題歸因：
   - 概念不熟
   - 題意誤讀
   - 選項比較失敗
   - 情境判斷錯誤
   - 法規/治理記憶不足
4. 補知識卡。
5. 請 CKO 依錯題出 5 題變形題。

## CKO 批改啟動語

```text
/agents CKO
我剛做完 AIAP 官方公告試題，以下是錯題紀錄。請幫我：
1. 分類弱點
2. 補 3 張知識卡
3. 出 5 題同考點變形題
先不要給答案。
```
