# AIAP 初級課程

## 科目 1：人工智慧基礎概論

來源：[[sources/guide-junior-subject-1-ai-basics.pdf]]

### Module 1：人工智慧概念

必會：

- AI 的定義與分類。
- 分析型 AI、預測型 AI、生成型 AI 的差異。
- AI 治理概念：透明性、公平性、隱私性、安全性、問責性。

練習方向：

- 給一個產業情境，判斷適合的 AI 技術。
- 給一個治理問題，判斷違反哪個倫理原則。

### Module 2：資料處理與分析概念

必會：

- 資料型態：結構化、半結構化、非結構化。
- 大數據特性：Volume、Velocity、Variety、Veracity、Value。
- 資料整理與分析流程。
- 資料隱私與安全。

練習方向：

- 判斷資料來源與資料型態。
- 分析資料品質、偏誤、隱私風險。

### Module 3：機器學習概念

必會：

- 監督式、非監督式、強化學習。
- 分類、迴歸、分群、異常偵測。
- 常見模型：線性模型、決策樹、KNN、SVM、神經網路。
- 訓練、驗證、測試與交叉驗證。

練習方向：

- 根據任務選擇學習方式。
- 判斷模型對資料尺度、標註資料、時間序列切分的需求。

### Module 4：鑑別式 AI 與生成式 AI 概念

必會：

- 鑑別式 AI：判斷、分類、預測。
- 生成式 AI：文字、圖像、語音、影片生成。
- 兩者的整合應用。
- 生成式 AI 的風險：幻覺、偏見、資安、隱私、版權。

練習方向：

- 判斷某應用屬於鑑別式、生成式或混合式。
- 根據風險情境提出控管措施。

## 科目 2：生成式 AI 應用與規劃

來源：[[sources/guide-junior-subject-2-generative-ai.pdf]]

### Module 5：No Code / Low Code 概念

必會：

- No Code 與 Low Code 的差異。
- 優勢：快速開發、降低技術門檻、支援流程自動化。
- 限制：客製化彈性、治理、資安、維護性。

練習方向：

- 根據企業需求判斷是否適合 No Code / Low Code。
- 判斷哪些場景需要專業開發介入。

### Module 6：生成式 AI 應用領域與工具使用

必會：

- 常見應用：文字生成、摘要、客服、知識管理、影像生成、簡報、程式輔助。
- Prompt 基本能力：角色、任務、背景、限制、輸出格式。
- 工具使用風險與驗證方式。

練習方向：

- 改寫 prompt 以提升輸出品質。
- 判斷生成式 AI 產出是否需要人工驗證。

### Module 7：生成式 AI 導入評估規劃

必會：

- 導入評估：業務價值、資料可得性、風險、成本、組織成熟度。
- 導入規劃：PoC、試點、擴散、治理、教育訓練。
- 風險管理：資料外洩、錯誤輸出、偏見、著作權、供應商依賴。

練習方向：

- 給企業案例，判斷導入階段與優先工作。
- 建立一份簡短 AI 導入計畫。

## 初級通關策略

- 科目 1：重點在概念辨識與情境判斷。
- 科目 2：重點在工具導入、流程規劃與風險控管。
- 官方公告試題至少做兩輪：第一輪計分，第二輪只看錯題與猶豫題。
