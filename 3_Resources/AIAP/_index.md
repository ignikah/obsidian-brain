# AIAP

> AI 應用規劃師證照學習系統 — 以官方學習指引與公告試題為核心，建立「學習 → 練習 → 錯題 → 複習」循環。

## 成員

- [[sources-index]] — 官方來源、下載 PDF 與使用說明
- [[course-roadmap]] — 初級到中級的總課程路線
- [[junior-course]] — 初級 AI 應用規劃師課程
- [[intermediate-course]] — 中級 AI 應用規劃師課程
- [[practice-system]] — 官方公告試題與 CKO 出題練習流程
- [[practice/diagnostic-junior]] — 初級診斷測驗
- [[templates/wrong-question-log]] — 錯題紀錄模板

## 子目錄

- `sources/` — 官方 PDF 原始檔，包含學習指引、勘誤表、公告試題
- `practice/` — 自製練習與診斷測驗
- `templates/` — 學習追蹤與錯題模板

## 使用方式

1. 先讀 [[course-roadmap]]，確認考試目標與時間。
2. 初級優先讀 [[junior-course]]，每週搭配官方公告試題練習。
3. 每次練習後用 [[templates/wrong-question-log]] 記錄錯題。
4. 請 CKO 依錯題紀錄出下一輪 5 題測驗。
