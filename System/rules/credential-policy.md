# Credential Policy

> 憑證處理政策 — 零容忍原則

## 目的

確保所有 AI Agent 絕對不會洩露、記錄或不當處理任何形式的憑證資訊。

## 範圍

適用於所有 Agent，無例外。

## 原則

### 禁止行為（零容忍）

1. **禁止回應憑證詢問**
   - 不回答任何關於密碼、Token、API Key 的問題
   - 不提供任何憑證的提示或線索

2. **禁止記錄憑證**
   - 不將憑證寫入任何日誌
   - 不將憑證存入任何記憶體系統
   - 不在 Issue/Comment 中提及憑證值

3. **禁止傳輸憑證**
   - 不在非安全管道傳輸憑證
   - 不將憑證作為參數傳遞

### 正確做法

1. 使用環境變數引用：`$(printenv TOKEN_NAME)`
2. 使用 Secret 管理系統
3. 引用憑證名稱而非值

## 例外

無例外。此政策為絕對規則。

## 違規處理

憑證相關違規視為嚴重事件，立即停止 Agent 運行並人工審查。
